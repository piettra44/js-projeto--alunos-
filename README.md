# 💻 Currículo Web - Portfólio

Este projeto é um **currículo online desenvolvido para a web**, criado com o objetivo de apresentar minhas informações, habilidades e alguns projetos de forma moderna e acessível.

🔗 **Acesse o site:**  
https://piettra44.github.io/CV-pietra/

---

## 📌 Sobre o projeto

O site funciona como um **currículo digital**, onde apresento meu perfil como desenvolvedora e algumas das minhas principais habilidades na área de tecnologia.

Nele você pode encontrar:

- 👩‍💻 Informações sobre mim
- 🧠 Minhas habilidades em desenvolvimento
- 📂 Alguns projetos que desenvolvi
- 📞 Formas de contato

---

## 🛠 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

- **HTML5** – estrutura da página  
- **CSS3** – estilização e responsividade  
- **JavaScript** – interatividade da interface

---

## 🎯 Objetivo

O objetivo deste projeto foi **praticar desenvolvimento front-end** e criar um **currículo online interativo**, que permita apresentar minhas habilidades e projetos de forma mais visual e profissional.

---

## 📱 Responsividade

O site foi desenvolvido para funcionar em diferentes dispositivos:

- 💻 Computadores
- 📱 Celulares
